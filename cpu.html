What is the CPU

The CPU, also known as the Central Processing Unit, uses electrical currents to represent 1 when on and 0 when off. CPUs are made up of mostly silicon, which gives it it’s ability to sometimes conduct electricity and sometimes to not, since it is a semi-conductor. The CPU then sends this information made up of binary numbers to different components, like the GPU, whose job is to decode the info then display it onto the moniter for the user to see.

Intel CPUs

All Intel CPUs that are not server-grade have integrated graphics, or also called APUs. This means most consumer chips can be used without a graphics card.

Code Names

CPUs have different generations. Each generation is different in it’s own way, bringing a performance boost with each generation. Each generation is given it’s own code name. The compatibility with previous generation hardware is not guaranteed, so it is important to research before buying. Here is a list of Intel’s code names and their corresponding generation number:

Code Name Generation
Westmere 1st gen
Sandy Bridge 2nd gen
Ivy Bridge 3rd gen
Haswell 4th
Broadwell 5th
Skylake 6th
Kaby Lake 7th
Coffee Lake 8th

Each generation’s naming scheme corresponds to it’s generation number. For example, i7 7700k is a 7th gen processor. First gen drops the starting number, like i7 640M(List of Intel CPU microarchitectures)
